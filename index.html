HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It is important to always remember to fill out the metadata in a page.
  • A particularly important point of metadata is the viewport, which sizes the page to fit varied screens.
  • There are many tags in HTML, but all of them use the angular bracket as demarkation.
